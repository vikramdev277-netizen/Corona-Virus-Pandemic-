# 🦠 COVID-19 Pandemic Data Analysis & Tracker

A Python-based data analysis project that processes state-wise daily COVID-19 data to uncover infection trends, regional patterns, and pandemic progression insights across India.

---

## 📌 Project Overview

This project analyzes real-world COVID-19 state-wise daily data to track the spread of the pandemic across Indian states. It processes structured health data to generate insights on infection rates, recovery trends, and regional hotspots — demonstrating practical skills in data analysis, cleaning, and visualization.

---

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| Python | Core programming language |
| Pandas | Data loading, cleaning, and transformation |
| NumPy | Numerical computations |
| Matplotlib / Seaborn | Data visualization and trend charts |
| Streamlit (app.py) | Interactive web dashboard |
| Jupyter Notebook | Exploratory data analysis |

---

## ✨ Features

- **State-Wise Analysis** – Breaks down COVID-19 cases, recoveries, and deaths by Indian state
- **Daily Trend Tracking** – Tracks day-over-day infection growth and recovery rates
- **Regional Hotspot Detection** – Identifies states with highest case loads and growth rates
- **Interactive Dashboard** – Streamlit-powered web app for dynamic data exploration
- **Data Cleaning Pipeline** – Handles missing values, duplicates, and inconsistent date formats

## 🚀 How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/vikramdev277-netizen/Corona-Virus-Pandemic-.git
   cd Corona-Virus-Pandemic-
   ```

2. **Install dependencies**
   ```bash
   pip install pandas numpy matplotlib seaborn streamlit
   ```

3. **Run the analysis script**
   ```bash
   python "corona virus pandemic.py"
   ```

4. **Launch the interactive dashboard**
   ```bash
   streamlit run app.py
   ```
